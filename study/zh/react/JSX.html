# JSX

# 1、jsx是什么?

  • React 使用 JSX 来替代常规的 JavaScript
  • JSX 是一个看起来很像 XML 的 JavaScript 语法扩展
  • 我们不需要一定使用 JSX，但它有以下优点

  1. JSX 执行更快，因为它在编译为 JavaScript 代码后进行了优化

  2. 它是类型安全的，在编译过程中就能发现错误

  3. 使用 JSX 编写模板更加简单快速
